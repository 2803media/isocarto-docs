# Édition d’un fichier

Cette page explique comment modifier un fichier déjà importé dans Isocarto.  
Depuis votre tableau de bord, chaque fichier dispose d’un menu d’actions permettant notamment de le renommer et de consulter ses informations principales.

## Renommer un fichier

Pour renommer un fichier :

1. Rendez-vous dans **Tableau de bord → Fichiers**.
2. Chaque fichier est affiché dans une boîte, accompagnée de son nom actuel.
3. À droite de cette boîte, cliquez sur le bouton **⋮** (trois points verticaux).
4. Sélectionnez **Renommer le fichier** dans le menu déroulant.
5. Saisissez le nouveau nom souhaité, puis confirmez.

![alt text](/img/edit-fichier.jpg)

Le fichier sera immédiatement mis à jour et le nouveau nom sera visible dans la liste.

## Informations disponibles dans la boîte du fichier

Chaque fichier affiché dans votre tableau de bord contient plusieurs informations importantes :

### Nombre d’adresses géocodées

Indique combien d’adresses ont été localisées automatiquement à partir de votre fichier.

### Nombre d’adresses avec coordonnées GPS

Affiche combien d’entrées contiennent déjà des coordonnées (latitude/longitude) fournies par vos soins.

### Nombre total d’entrées

Correspond au nombre total de lignes présentes dans le fichier importé.  
Cela inclut toutes les adresses ou données unitaires.

### Date et heure d’import

Permet de savoir à quel moment le fichier a été ajouté dans Isocarto.  
Cette information est utile pour suivre l’historique de vos imports ou identifier les fichiers les plus récents.
