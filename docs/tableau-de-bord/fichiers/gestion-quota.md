# Gestion du quota de fichiers

Dans Isocarto, chaque organisation dispose d’un quota global de fichiers et de données ajoutables, partagé entre tous les membres de l’abonnement.

## Comprendre le quota

Le quota représente la quantité totale de données que l’organisation peut stocker. Ce quota est réparti entre le propriétaire et les membres.

## Visualisation du quota

Dans l’interface, vous pouvez voir :

- **Votre propre utilisation** : la quantité de fichiers que vous avez ajoutés.
- **L’utilisation des autres membres** : représentée de manière anonymisée, afin de préserver la vie privée.

Chaque utilisateur voit ainsi une barre de progression indiquant le quota global, avec une couleur distincte pour sa propre utilisation et une autre couleur pour les autres membres.

## Gérer le quota

Si l’organisation atteint la limite de quota :

- [**Suppression de fichiers**](supprimer-fichier) : il faudra supprimer certains fichiers pour libérer de l’espace.
- **Demande de nettoyage** : les autres membres peuvent également être sollicités pour libérer de l’espace.

## Augmenter le quota

Si l’organisation est éligible à un abonnement supérieur, le quota sera automatiquement augmenté en fonction des limites de ce nouvel abonnement.
