# Ajouter une nouvelle carte

Cette page explique comment créer une nouvelle carte dans votre espace de travail sur **Isocarto**.

## Étape 1 : Accéder à l’ajout de carte

Dans la page **Vos cartes** dans votre tableau de bord, cliquez sur la **carte avec le symbole “+”**, intitulée **_Nouvelle carte_**.  
Ce bouton permet de créer rapidement une nouvelle carte dans votre compte.

![Bouton 'Nouvelle carte' avec icône + pour créer une nouvelle carte](/img/add-carte.jpg)

## Étape 2 : Création de la carte

Une fois le clic effectué, vous êtes immédiatement redirigé dans l’application sur une **nouvelle carte vide**, sans aucune zone préexistante.

## Étape 3 : Ajouter des zones

Votre carte est maintenant prête à être personnalisée selon vos besoins.  
Vous pouvez ajouter autant de zones que nécessaire, en fonction :

- de vos critères géographiques,
- de vos besoins d’analyse,
- ou des choix que vous définirez directement dans l’application.

Chaque zone pourra ensuite être configurée, éditée et exploitée dans vos analyses.
