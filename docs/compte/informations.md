# Informations du compte

L’onglet **Informations** vous permet de consulter et mettre à jour toutes vos données personnelles associées à votre compte Isocarto.

## Informations personnelles

Dans cette section, vous pouvez visualiser et modifier les informations enregistrées lors de votre inscription :

- **Nom**
- **Prénom**
- **Adresse e-mail**
- **Mot de passe**
- **Numéro de téléphone**

![Capture d'écran des informations du compte](/img/compte-preview.jpg)

Pour modifier une information, il suffit de mettre à jour le champ correspondant puis de cliquer sur **Enregistrer**.

## Mise à jour de l’adresse e-mail

Pour mettre à jour l'email associé à votre compte, vous devez :

1. Cliquez sur le bouton "changer l'email' pour accéder à la page dédiée.

![alt text](/img/changement-email.jpg)

:::warning[Attention]

La modification de l’adresse e-mail nécessite une validation de sécurité sur le nouvel email.

:::

2. Vous recevrez un message contenant un lien de confirmation.
3. Votre nouvelle adresse ne sera active qu’une fois cette confirmation effectuée.

## Mise à jour du mot de passe

Pour changer votre mot de passe, vous devrez :

1. Cliquez sur le bouton "changer le mot de passe" pour accéder à la page dédiée.

![alt text](/img/changement-mdp.jpg)

2. Saisir votre mot de passe actuel.
3. Saisir le nouveau mot de passe.
4. Confirmer votre choix.

Cette étape garantit la sécurité de votre compte.

## Suppression définitive du compte

:::warning[Attention]

**Attention : cette action est irréversible.**

:::

La section située en bas de la page permet de **supprimer entièrement votre compte Isocarto** ainsi que toutes les données associées.

![alt text](/img/del-compte.jpg)

En supprimant votre compte :

- Toutes vos données personnelles seront effacées.
- Toutes vos cartes, zones, fichiers et configurations seront supprimés définitivement.
- Vos abonnements en cours seront **annulés immédiatement**.
- Vous perdrez instantanément l’accès aux fonctionnalités associées.

Pour finaliser cette action, une confirmation est nécessaire afin d’éviter toute suppression accidentelle.
