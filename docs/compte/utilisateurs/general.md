# Gestion des utilisateurs

L’onglet **Utilisateurs** vous permet de gérer les membres associés à votre organisation Isocarto.  
Cet espace centralise toutes les informations concernant les membres, leurs rôles, ainsi que les actions d’administration disponibles.

## Gestion des membres de l'organisation

Dans cette section, vous pouvez :

- Consulter la **liste des utilisateurs** appartenant à votre organisation.
- Voir pour chaque membre :
  - **Nom et prénom**
  - **Adresse e-mail**
  - **Rôle** (Propriétaire ou Member)
  - **Date d’ajout** dans l’organisation
- Déterminer rapidement qui possède les droits d’administration.

![alt text](/img/member-1.jpg)

## Rôles dans l’organisation

Trois rôles sont possibles :

### **Propriétaire**

- Il s’agit du propriétaire de l’organisation.
- Dispose de **tous les droits**, notamment :
  - Gestion de l’abonnement.
  - Gestion des moyens de paiement.
  - Accès aux factures.
  - Ajout ou suppression de membres.
  - Paramétrage avancé.

### **Administrateur**

- Peut utiliser toutes les fonctionnalités incluses dans l’abonnement de l’organisation.
- Peut inviter et gérer les membres.
- Ne peut **pas** gérer :

  - [L'abonnement](../abonnement/general)
  - [La facturation](../factures/general)
  - [Les paramètres de l’organisation](../informations)
  - [Les données à afficher dans les rapports](../../tableau-de-bord/rapport)

### **Member**

- Peut utiliser toutes les fonctionnalités incluses dans l’abonnement de l’organisation.
- Ne peut **pas** gérer :
  - L’abonnement.
  - La facturation.
  - Les membres.
  - Les paramètres de l’organisation.

## Actions disponibles

Selon votre rôle, vous pouvez :

### Propriétaire

- **Inviter un utilisateur**
- **Supprimer un membre**
- **Révoquer une invitation en attente**
- **Changer le rôle** d'un membre

![alt text](/img/member-3.jpg)

### Administrateur

- **Inviter un utilisateur**
- **Supprimer un membre**
- **Révoquer une invitation en attente**

### Member

- Accéder aux ressources partagées par l’organisation

## Supprimer un membre

La suppression d'un membre supprimera tous ses accès ainsi que ses données. Cela inclut l'accès aux cartes, aux données et aux paramètres personnels.

![alt text](/img/member-6.jpg)

:::tip[Information]

La révocation d'un membre mettra à jour votre abonnement automatiquement avec une réduction correspondant au coût mensuel du membre supprimé.

:::
