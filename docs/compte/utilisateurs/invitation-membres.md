# Invitation de membres

Cette page détaille le processus permettant d’inviter de nouveaux membres à rejoindre votre organisation Isocarto.  
Seul le **propriétaire (Owner)** ou un **Administrateur** de l’organisation peut réaliser cette opération.

## Inviter un nouveau membre

:::warning[Attention]

Un membre invité **ne peut pas** avoir déjà un **compte actif sur Isocarto**. Si l'utilisateur dispose déjà d'un compte, il devra d'abord se clôturer sont compte pour pouvoir rejoindre l'organisation via une invitation spécifique.

:::

:::tip[Information]

L'ajout de membre à votre organisation impliquera une facturation supplémentaire conformément aux tarifs en vigeur dans les abonnements.

:::

Pour inviter un utilisateur :

1. Rendez-vous dans **Compte → Utilisateurs**.
2. Saisissez l’adresse e-mail du membre à inviter.
3. Validez l’envoi de l’invitation.

![Formulaire d'invitation d'un nouveau membre avec champ d'email et bouton d'envoi d'invitation](/img/member-5.jpg)

Une entrée apparaît alors dans la liste des invitations en attente.

## E-mail d’invitation envoyé au membre

Lorsqu’une invitation est envoyée, le futur membre reçoit un e-mail contenant :

- Le **nom de l’organisation**
- L'identité du propriétaire ayant envoyé l’invitation
- Un bouton ou lien **Accepter l'invitation**
- Les informations essentielles sur le partage d’un abonnement

![Aperçu de l'email d'invitation reçu par le nouveau membre avec les détails de l'organisation et le bouton 'Accepter l'invitation'](/img/member-2.jpg)

Ce message permet au membre d’accéder directement à la page d’enrôlement Isocarto.

## Page d’enrôlement

En cliquant sur le lien de l’e-mail, l’invité est redirigé vers la **page d’enrôlement** simplifiée.  
Il devra :

1. Renseigner son nom et prénom
2. Choisir un mot de passe pour se connecter

![Formulaire d'enrôlement avec les champs Nom, Prénom, Mot de passe et bouton de confirmation](/img/member-4.jpg)

Une fois ces étapes validées, il devient officiellement membre de l’organisation.

## Révocation d’une invitation

Le propriétaire peut à tout moment :

- **Annuler une invitation en attente**

![Menu contextuel d'un membre avec l'option 'Révoquer l'invitation' mise en évidence](../../../static/img/member-7.jpg)

Cette action est accessible depuis la liste des utilisateurs via le menu d’options.

## Limites et conditions

- Le nombre de membres autorisés dépend du **plan d’abonnement** de l’organisation.
- Si la limite est atteinte, il sera nécessaire d’effectuer un **upgrade** de plan pour inviter davantage de membres.
- Les membres ajoutés consomment le quota de l’organisation pour les téléversements de fichiers (selon les limites de l'abonnement).
