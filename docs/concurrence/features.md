# Comparaison des fonctionnalités

Ce document compare les principales capacités des plateformes de géomarketing et d’analyse de zones : **Isocarto**, **Smappen** et **Geo‑Business**.

| **Fonctionnalité**                         | **Isocarto**                                                          | **Smappen**                                                               | **Geo‑Business**                                                                   |
| ------------------------------------------ | --------------------------------------------------------------------- | ------------------------------------------------------------------------- | ---------------------------------------------------------------------------------- |
| **Création de zones**                      | Isochrones, isodistances, administratives, manuelles                  | Isochrones, isodistance, zones administratives, manuelles                 | Isochrones, distances, manuelles, sélections admin (codes postaux, communes, IRIS) |
| **Nombre de zones**                        | Selon plan tarifaire (3 zones en gratuit, illimité en plan supérieur) | Limité à 10 zones (gratuit), illimité en plans supérieurs                 | 15 zones (Starter) ou illimité selon plan                                          |
| **Cartes interactives**                    | Oui (interface web)                                                   | Oui (interface web)                                                       | Oui (avec sélection de types de zones)                                             |
| **Données démographiques**                 | Données INSEE avancées selon plan                                     | Démographie selon plans (population, revenus, etc.)                       | Données socio‑démographiques accessibles (plus de 90 indicateurs)                  |
| **Points d’intérêt / entreprises**         | Base SIRENE incluse dans plans supérieurs                             | Recherche entreprises & services (limité ou illimité selon plan)          | Recherches d’entreprises personnalisées                                            |
| **Import de données externes**             | Oui (points personnalisés selon plan)                                 | Oui (Excel/HubSpot, points selon plan)                                    | Oui (import jusqu’à 50 000 ou illimité selon plan)                                 |
| **Export de zones / rapports**             | GeoJSON, PNG, XLS, PDF (selon plan)                                   | KML, Excel, Word, images HD (selon plan)                                  | Export tableau/rapports, images, formats Excel/PDF                                 |
| **Analyse de marché géomarketing**         | Données détaillées (emploi, logement, revenu…)                        | Indicateurs personnalisés & rapports ELM (Pro+)                           | Indicateurs socio‑économiques poussés (+ packs)                                    |
| **Indicateurs personnalisés / métriques**  | Certaines analytiques intégrées                                       | Dashboard métriques personnalisées                                        | Création d’indicateurs personnalisés (Standard)                                    |
| **Multi‑utilisateurs / collaboration**     | Selon plan (ajout d’utilisateurs)                                     | Plan Avancé + utilisateurs supplémentaires                                | Ajout utilisateurs (Standard)                                                      |
| **Rapports automatisés (ELM / synthèses)** | Oui (PDF/Excel détaillés)                                             | Oui (Word / Excel automatisés - Pro+)                                     | Export de rapports et synthèses                                                    |
| **Interface intuitive / facile**           | Oui, orientée géomarketing                                            | Oui, très intuitive (focus PME)                                           | Oui, plateforme riche mais professionnelle                                         |
| **Catalogue de données intégré**           | Données sociodémographiques et immobilières (selon plan)              | Données démographiques & entreprises (France, BE, USA, UK, CA selon plan) | +300 indicateurs socio‑économiques et données métiers                              |
| **Données uniques**                        | Plan cadastral                                                        | -                                                                         | Plan cadastral                                                                     |
|                                            | Propriétaires personnes morales                                       | -                                                                         | Propriétaires personnes morales                                                    |
|                                            | Transactions immobilières                                             | -                                                                         | Transactions immobilières                                                          |
|                                            | Inventaire biophysique                                                | -                                                                         | -                                                                                  |
|                                            | Métrage automatique des parcelles                                     | -                                                                         | -                                                                                  |
|                                            | PLU (Plan Local d’Urbanisme)                                          | -                                                                         | -                                                                                  |
|                                            | Registre parcellaire graphique                                        | -                                                                         | -                                                                                  |
|                                            | Réseaux de transport                                                  | -                                                                         | -                                                                                  |
|                                            | Risques d'inondation (TRI)                                            | -                                                                         | -                                                                                  |
|                                            | Taux de cambriolages                                                  | -                                                                         | -                                                                                  |
|                                            | Zones patrimoine et monuments historiques                             | -                                                                         | -                                                                                  |
| **Export formats SIG / vecteurs**          | GeoJSON                                                               | KML                                                                       | GeoJSON / formats SIG selon plan                                                   |
