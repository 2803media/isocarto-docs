# Ajout de données métier dans Isocarto

Dans Isocarto, il est possible d'ajouter des données provenant de fichiers Excel ou CSV pour enrichir l'analyse de votre zone d'échalandise. Ces données métier permettent d'avoir une vision plus approfondie de votre zone d'échalandise.

Pour ajouter des données métier, rendez-vous dans la barre latérale de gauche et cliquez sur **"Données métier"**. Ensuite cliquez sur **"Gérez vos données"**.

![onglet données métier](/img/datas-business-5.jpg)

## Pour ajouter un nouveau jeu de données

Vous pouvez importer des fichiers au format :

- **Excel (.xlsx)**
- **CSV (.csv)**

Les fichiers doivent contenir des données géolocalisables ou exploitables par zone (adresse, coordonnées GPS, identifiants, valeurs numériques, etc.), afin de permettre leur agrégation et leur restitution cartographique.

Une fois le fichier importé, Isocarto associe automatiquement les données aux zones correspondantes.
