# Données métier

L’onglet **Données métiers** permet d’exploiter vos **propres fichiers de données** (Excel ou CSV) directement dans Isocarto, afin d’obtenir une analyse **zone par zone**, parfaitement adaptée à votre activité.

Ces données sont idéales pour croiser vos informations internes (chiffre d’affaires, performances, indicateurs commerciaux, etc.) avec les zones d’analyse définies sur la carte.

## Accès à l’onglet Données métiers

Rendez-vous dans la barre latérale gauche de l’application, puis cliquez sur **"Données métiers"**.

![onglet données métier](/img/datas-business-1.jpg)

Cet espace est dédié à la gestion, la visualisation et l’analyse des données issues de vos fichiers importés.

## Présentation d’un jeu de données

Dans le bloc de présentation de chaque donnée, vous retrouvez :

- la **couleur utilisée** pour l’affichage sur la carte,
- le **nom du jeu de données**,
- un menu accessible via les **trois points**, permettant d’interagir avec la donnée.

Sous ce bloc, sont affichées des informations complémentaires :

- le **nombre de résultats dans la zone active**.
- Vous pouvez afficher ou masquer les données sur la carte avec le toggle vert.

:::tip[Indicateur du nombre de résultats]

Le nombre de résultats affichés constitue un **indicateur clé** :  
il permet de comprendre rapidement combien de points d’intérêt sont présents **dans la zone actuellement analysée**, en lien avec les données métier ajoutées.

:::

## Visualisation des données sur la carte

Les données métiers sont représentées directement sur la carte, sous forme de **points** ou de d'aggrégations en clusters.

Pour chaque jeu de données, il est possible d'activer ou désactiver l’affichage avec le toggle vert.

## Construction d’indicateurs personnalisés

À partir des données contenues dans votre fichier, Isocarto permet de créer des **indicateurs métiers**.

Ces indicateurs sont calculés **zone par zone**, en fonction des valeurs numériques présentes dans votre fichier.

![onglet données métier](/img/datas-business-2.jpg)

Les opérations disponibles incluent notamment :

- **Somme** : Additionne toutes les valeurs numériques de la colonne sélectionnée pour les points présents dans la zone d'analyse.
- **Moyenne** : Calcule la moyenne arithmétique des valeurs de la colonne sélectionnée pour les points présents dans la zone d'analyse.
- **Ecart type** : Mesure la dispersion des valeurs par rapport à leur moyenne, indiquant la variabilité des données dans la zone d'analyse.

![onglet données métier](/img/datas-business-3.jpg)

Les indicateurs s’adaptent dynamiquement aux colonnes du fichier importé, offrant une grande flexibilité d’analyse.

## Gestion des pop-ups

Les **pop-ups cartographiques** peuvent être activées ou désactivées sur les données métiers.

![onglet données métier](/img/datas-business-4.jpg)

Lorsqu’une pop-up est affichée :

- Elle présente **l’intégralité des données disponibles** pour l’entité sélectionnée au survol du point sur la carte
- Les champs affichés correspondent aux colonnes du fichier importé

![onglet données métier](/img/datas-business-6.jpg)

Cela permet une lecture complète et détaillée des informations directement depuis la carte et des rapports.

## Renommage et lisibilité des données

Pour améliorer la compréhension et la lisibilité :

- Les jeux de données peuvent être **renommés**
- Les intitulés peuvent être adaptés à votre vocabulaire métier

Cette personnalisation facilite le travail collaboratif et l’interprétation des cartes. Par défaut le nom du jeu de données est celui du fichier importé.

## Téléchargement des données

Il est possible de **télécharger vos données métiers** au format Excel en fonction de la zone active.

Cela facilite :

- Les analyses externes
- Le partage des données
- L’archivage des résultats

## Personnalisation de l’apparence

Pour chaque jeu de données métiers, vous pouvez :

- **Changer la couleur** d’affichage sur la carte
- Ajuster la visibilité

Ces actions permettent de garder une carte claire et pertinente, même avec de nombreuses couches actives.

## Suppression des données

À tout moment, vous pouvez :

- Supprimer un jeu de données métiers de la carte (cela ne supprimera pas le jeu de données de votre compte)
- Mettre à jour vos analyses avec des fichiers plus récents
