# Importer des zones depuis Géoportail

Depuis Géoportail, vous pouvez exporter une zone isochrone calculée, directement compatible avec l’import dans Isocarto. L'export des zones isochrones depuis Géoportail se fait zone par zone.

## Étapes pour exporter depuis Géoportail

1. Connectez-vous à votre compte Géoportail.
2. Après avoir créé votre isochrone et calculé la zone, affichez-la sur la carte.

![import de zones - geoportail](/img/geoportail-export-1.jpg)

3. En bas à droite, sous la couche appelée par défaut **isocurve**, cliquez sur le bouton **Exporter**.

![import de zones - geoportail](/img/geoportail-export-2.jpg)

4. Le fichier GeoJSON sera généré, prêt à être importé dans Isocarto.

:::tip[Astuce]

Vérifiez bien les détails de l’isochrone avant l’export pour garantir la précision de votre import dans Isocarto.

:::
