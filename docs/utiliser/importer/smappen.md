# Importer des zones depuis Smappen

Pour importer des zones depuis Smappen, connectez-vous à votre compte Smappen et ouvrez la carte contenant les zones à exporter.

## Étapes d'exportation

1. Sélectionnez la carte que vous souhaitez exporter.
2. Dans la barre en haut de la page, cliquez sur les trois petits points situés dans la bannière.
3. Choisissez l’option **Exporter (KML)**.

![import de zones - smappen](/img/smappen-export-1.jpg)

4. Le fichier KML sera généré, contenant toutes les zones de cette carte.

Si vous avez plusieurs cartes, répétez l’opération pour chaque carte. L’export se fait en quelques secondes, et le fichier sera prêt à être importé dans Isocarto.

:::tip[Astuce]

Pensez à organiser vos cartes Smappen avant l’export pour une gestion optimisée de vos zones.

:::
