# Utiliser Isocarto dans ChatGPT

Isocarto peut être utilisé directement dans ChatGPT grâce au plugin Isocarto. Cette intégration permet de créer et visualiser des zones de chalandise, d'interroger les données locales et de préparer une analyse géomarketing complète en langage naturel.

Le plugin donne à ChatGPT accès aux cartes, aux zones et aux données territoriales d'Isocarto. Il peut ainsi mobiliser des informations locales précises que ChatGPT ne possède pas seul, puis les interpréter selon votre projet d'implantation ou votre étude de marché.

OpenAI appelait auparavant ces intégrations des apps ChatGPT. Elles sont désormais distribuées comme plugins dans le Plugin Directory.

## Installer le plugin Isocarto

1. Ouvrez la page de ressources : [Isocarto dans ChatGPT](https://isocarto.fr/chatgpt).
2. Cliquez sur le bouton d'installation, ou ouvrez directement [le plugin Isocarto dans ChatGPT](https://chatgpt.com/plugins/plugin_asdk_app_6a06ebd93b688191aab095f9511c9f28).
3. Connectez-vous à votre compte ChatGPT si nécessaire.
4. Lancez une conversation avec Isocarto.

![Plugin Isocarto dans ChatGPT](/img/Chatgpt.jpg)

Une fois le plugin ouvert, vous pouvez poser une question comme vous le feriez à un analyste : indiquez le lieu, le type de zone, l'objectif de votre étude et les critères qui comptent pour votre projet.

## Exemples de demandes

Vous pouvez par exemple demander :

- « Analyse le potentiel commercial d'une implantation de boulangerie à Nantes. »
- « Compare deux emplacements pour une franchise de restauration rapide à Lyon. »
- « Aide-moi à préparer une étude locale de marché pour une ouverture de salle de sport. »
- « Analyse la population, les âges, les revenus, les ménages, l'emploi et le logement de cette zone. »
- « Construis le profil territorial de ces secteurs, segmente-les et classe-les selon leur potentiel. »
- « Quelles données dois-je regarder pour évaluer une zone de chalandise ? »
- « Résume les points forts et les risques d'un local situé à proximité d'une gare. »

Plus votre demande est précise, plus la réponse sera utile. Pensez à préciser l'activité, l'adresse ou la ville, le format de zone souhaité, vos concurrents connus et le type de décision à prendre.

## Ce que le plugin peut vous aider à faire

Le plugin Isocarto dans ChatGPT peut vous accompagner pour :

- créer des zones isochrones, isodistantes, circulaires ou administratives ;
- afficher vos cartes et retrouver les zones déjà enregistrées ;
- compter des commerces, entreprises, services et équipements issus des bases BPE, OpenStreetMap et SIRENE ;
- transformer une question métier en analyse géomarketing ;
- analyser les indicateurs démographiques, économiques et résidentiels d'une zone ;
- préparer une comparaison entre plusieurs emplacements ;
- réaliser un profilage territorial à partir de données locales agrégées ;
- segmenter, classer et prioriser des zones selon leur potentiel ;
- structurer une synthèse pour un dossier d'implantation, une franchise ou une étude locale de marché.

Il est particulièrement utile en amont d'une analyse, lorsque vous ne savez pas encore quelles données regarder, mais aussi pour produire directement une première synthèse exploitable dans un document de décision.

## Réaliser une étude de marché dans ChatGPT

Isocarto permet à ChatGPT de mobiliser plus de 50 indicateurs territoriaux pour étudier une zone de chalandise :

- population et répartition par tranche d'âge ;
- répartition femmes-hommes ;
- composition et taille des ménages ;
- niveau de vie, pauvreté et déciles de revenus ;
- catégories socio-professionnelles, population active et chômage ;
- maisons, logements collectifs, propriétaires et locataires ;
- résidences principales, secondaires ou vacantes ;
- taille, occupation et ancienneté du parc de logements.

Lorsque les données sont disponibles, ChatGPT peut comparer les résultats de la zone aux références nationales, identifier les écarts significatifs et faire ressortir les caractéristiques dominantes du territoire.

### Profilage territorial

Le profilage territorial consiste à synthétiser les caractéristiques démographiques, économiques, résidentielles et commerciales d'une zone. Il s'appuie sur des données territoriales agrégées, et non sur des données personnelles.

Vous pouvez par exemple demander à ChatGPT d'identifier un profil familial, étudiant, résidentiel, actif, aisé ou populaire, puis de préciser les opportunités et les points de vigilance associés à votre activité.

### Segmentation stratégique

La segmentation stratégique permet de comparer plusieurs zones avec une même grille de lecture. ChatGPT peut les différencier selon leur population, leur potentiel de consommation, leur structure sociale, leur marché de l'emploi, leur habitat ou leur environnement commercial.

Cette approche permet notamment de :

- classer plusieurs emplacements potentiels ;
- repérer les zones partageant un profil similaire ;
- adapter un concept commercial à la clientèle locale ;
- prioriser les secteurs à approfondir dans Isocarto.

## Données disponibles selon l'abonnement

Les informations transmises à ChatGPT respectent automatiquement les fonctionnalités incluses dans l'abonnement Isocarto de l'utilisateur. Dans la gamme standard :

| Abonnement | Analyse territoriale disponible |
| --- | --- |
| Starter | Population |
| Plus | Population, tranches d'âge et répartition femmes-hommes |
| Pro | Données précédentes, ménages, catégories socio-professionnelles et chômage |
| Enterprise | Données précédentes, revenus et logement |

Lorsqu'une rubrique n'est pas comprise dans l'abonnement, ChatGPT présente d'abord toutes les informations disponibles, puis signale les données complémentaires accessibles avec une offre supérieure. Les données verrouillées ne sont jamais communiquées.

## Révoquer les tokens

Si vous souhaitez révoquer les tokens générés par le plugin Isocarto dans ChatGPT, vous pouvez le faire depuis les paramètres de votre compte ChatGPT en cliquant sur "Déconnecter" ou "Disconnect" dans la section "Connected apps".

![révoquer les tokens](/img/chatgpt-revoke-openai.jpg)

Mais aussi directement depuis Isocarto dans l'onglet ChatGPT de la page de "Gestion de votre compte". C'est pratique s'il y a un bug quelconque ou si vous voulez simplement désactiver le plugin.

![révoquer les tokens](/img/chatgpt-revoke.jpg)

## Bonnes pratiques

Pour obtenir une réponse plus pertinente :

- donnez un contexte métier clair ;
- précisez le territoire étudié ;
- indiquez le niveau de détail attendu ;
- demandez une réponse structurée si vous préparez un rapport ;
- vérifiez les résultats importants dans la carte Isocarto.

Exemple de prompt complet :

> Je souhaite ouvrir une boutique de prêt-à-porter féminin à Rennes, près du centre-ville. Aide-moi à évaluer le potentiel de la zone, les indicateurs démographiques utiles, les points de vigilance et les données à vérifier dans Isocarto.

Pour une étude plus complète :

> Crée une zone de chalandise de 10 minutes autour de cette adresse, puis prépare son profil territorial. Analyse les données disponibles sur la population, les âges, les revenus, les ménages, les catégories socio-professionnelles, le chômage et le logement. Compare les résultats avec la France, identifie les points forts et les risques, puis propose une segmentation adaptée à mon projet.

## Limites à connaître

Les indicateurs disponibles dépendent de votre abonnement Isocarto. ChatGPT indique les rubriques auxquelles votre compte peut accéder et signale celles qui nécessitent une offre supérieure.

Pour approfondir une décision opérationnelle, vous pouvez ouvrir la carte Isocarto afin de consulter les zones, les couches de données et les rapports détaillés.
