# Identifier une zone similaire avec une zone de référence

L'outil **Zone similaire** d'Isocarto permet de trouver des zones comparables à une zone de référence, en France. Vous pouvez lancer cette analyse en quelques étapes simples.

## Étapes pour utiliser l'outil

1. Ouvrez l’onglet **Outils** dans le menu principal d’Isocarto.
2. Cliquez sur **Zones similaires**.
3. Cliquez sur **zone de référence**.
4. Sélectionnez une carte, puis **une zone que vous avez déjà créée**.
5. Affinez la recherche en utilisant les filtres : vous pouvez sélectionner des régions, départements ou typologies (hypercentre, centre-ville, périphérie).
6. Lancez l’analyse en cliquant sur le bouton dédié.

![Exemple de zone similaire](/img/zones-similaires-1.jpg)

L’outil va ensuite calculer un score de similarité et proposer jusqu’à 9 zones similaires classées par pertinence.

![Exemple de zone similaire](/img/zones-similaires-2.jpg)

## Affiner votre recherche

Après l’analyse, vous pouvez :

- Cliquer sur les zones directement dans la carte pour les visualiser.
- Affiner en fonction des profils (équilibrée, industrielle, etc.).
- Explorer les scores associés pour mieux comprendre les similarités.

:::tip[Astuce]

Utilisez ces zones similaires pour vos recherches de nouvel emplacement, études de marché ou pour comparer des zones stratégiques en France.

:::
