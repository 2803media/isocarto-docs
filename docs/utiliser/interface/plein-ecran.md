# Passer en mode plein écran

## Présentation

Isocarto offre un mode plein écran qui permet d’afficher la carte de façon immersive, sans distractions.

## Utilisation du bouton plein écran

Le bouton plein écran se trouve en haut à droite de la carte, sous les boutons de zoom.

![bouton full screen](/img/full-screen.jpg)

## Fonctionnement

1. **Cliquer sur le bouton plein écran** : En cliquant sur ce bouton, la carte passe en mode plein écran.

2. **Suppression des éléments perturbateurs** : En mode plein écran, la barre latérale et la barre de recherche disparaissent, laissant la carte totalement dégagée.

3. **Vue immersive** : Cette vue permet d’avoir une vision plus claire et plus détaillée de la carte, idéale pour une analyse approfondie.

## Conclusion

Le mode plein écran améliore considérablement l’expérience utilisateur et permet de se concentrer pleinement sur les données affichées.
