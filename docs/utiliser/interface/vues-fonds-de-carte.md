# Usages recommandés des fonds de carte

## Vue Satellite

![Vue satellite](/img/fond-sat.jpg)

La vue satellite est particulièrement utile pour :

- visualiser la réalité du terrain : bâtiments, toitures, parkings, zones industrielles ou commerciales ;
- analyser la densité bâtie ou l’évolution urbaine ;
- repérer des éléments non visibles sur une carte classique ;
- produire des visuels représentant fidèlement l’environnement.

:::tip[Source]

Isocarto utilise les orthophotos de l’IGN (BD ORTHO), reconnues pour leur précision et leur mise à jour régulière.

:::

## Vue Classique

![Vue classique](/img/fond-normal.jpg)

Le fond de carte classique est optimisé pour :

- les travaux d’analyse nécessitant une lecture claire et structurée ;
- les présentations ou rapports demandant un rendu propre et neutre ;
- la superposition de nombreuses couches (INSEE, SIREN, zones, fichiers, heatmaps) ;
- une navigation fluide et lisible.

:::tip[Information]

A l'inverse des concurrents, Isocarto utilise **son propre serveur de tuiles** basé sur les données OpenStreetMap, permettant une maîtrise complète du style et des éléments affichés.

:::

## Aucun fond de carte

![Vue sans fond](/img/fond-blanc.jpg)

Ce mode est particulièrement utile pour :

- les exports destinés aux rapports ou présentations ;
- mettre en avant uniquement les zones, données ou fichiers ;
- les analyses comparatives où le fond de carte pourrait perturber la lecture.
