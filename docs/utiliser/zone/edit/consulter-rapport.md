# Consulter un rapport

Cette page explique comment accéder et consulter un rapport dans l'interface d'Isocarto depuis le menu d'édition d'une zone.

## Accès au panneau de rapport

Pour consulter un rapport :

1. Localisez le bouton **Accéder au rapport** dans l'interface
2. Cliquez sur ce bouton

![button launch rapport](/img/access-report.jpg)

Le panneau de rapport s'ouvrira alors, affichant les informations détaillées du rapport généré pour la zone sélectionnée.

## Navigation dans le rapport

Une fois le panneau ouvert, vous pourrez :

- Parcourir les différentes sections du rapport
- Consulter les statistiques et analyses
- Télécharger le rapport si nécessaire

## Fermeture du panneau

Pour fermer le panneau de rapport, cliquez sur la croix (×) dans le coin supérieur droit du panneau ou en dehors de celui-ci.
