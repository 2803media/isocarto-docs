# Télécharger une zone

Isocarto permet d’exporter vos zones pour les utiliser dans d’autres outils SIG, applications internes ou présentations.

## Formats disponibles

- **GeoJSON** : idéal pour QGIS, Python, MapLibre, ou tout workflow géospatial. Pour visualiser rapidement un geojson vous pouvez utiliser [MapShaper](https://mapshaper.org/).
- **Image (PNG)** : utile pour les rapports, slides ou documents rapides. Cette option est détaillée dans la section suivante [Imprimer une carte](/utiliser/interface/impression-carte).

## Procédure

1. Ouvrez le panneau d’édition.
2. Cliquez sur **Télécharger le GEOJSON**.

![button add zone](/img/download-zone.jpg)

## Notes

- L’export GeoJSON contient le polygone exact de la zone et ses métadonnées.
