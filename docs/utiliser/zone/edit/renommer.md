# Renommer une zone

Pour faciliter l’organisation de vos cartes, chaque zone peut être renommée directement depuis son panneau d’édition.

<VimeoVideo videoId="1146032792" />

## Procédure

1. Ouvrez le panneau d’édition de la zone.
2. Cliquez sur le bouton **Renommer la zone**.
3. Saisissez le nouveau nom dans la fenêtre de dialogue.
4. Validez.

## Bonnes pratiques

- Utiliser des noms décrivant l’objectif de la zone
- Standardiser les noms si vous travaillez en équipe
