# Supprimer une zone

La suppression d’une zone est définitive.  
Elle efface la géométrie, les paramètres et les données associées (filtres, réglages, analyses locales...).

## Procédure

1. Ouvrez le panneau d’édition de la zone.
2. Cliquez sur **Supprimer**.

![button supprimer zone](/img/delete-zone.jpg)

3. Confirmez la suppression dans la fenêtre qui s’affiche.

![button supprimer zone](/img/delete-zone-2.jpg)

## Sécurité

- Une confirmation est obligatoire pour éviter toute suppression accidentelle.
- Si plusieurs zones sont liées à un même projet, les autres ne sont pas affectées.
