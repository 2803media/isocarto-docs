# Modifier une zone

Avec Isocarto vous pouvez modifier le type de tracé d'une zone existante, que ce soit une zone isochrone, isodistante, administrative ou manuelle.

<VimeoVideo videoId="1146032156" />

## Modifier une zone isochrone ou isodistante

Pour ces zones calculées automatiquement :

- vous pouvez ajuster le **temps de trajet** (ex : 10, 15, 20 minutes) ;
- ou modifier la **distance** (ex : 1 km, 5 km, 20 km).

Chaque modification relance automatiquement le calcul et met à jour le tracé.

## Modifier une zone manuelle

Les zones manuelles sont entièrement éditables :

- déplacer chaque sommet (vertex) ;
- ajouter ou supprimer des points ;
- ajuster la forme en glissant directement sur la carte.

Une fois terminé, cliquez sur **Enregistrer** pour valider les modifications.

## Modifier une zone administrative

Les zones administratives sont fixes et basées sur des limites officielles.  
On peut modifier :

- l’entité sélectionnée (Iris, commune, EPCI, département…) ;
- mais pas le tracé, qui est figé par définition
