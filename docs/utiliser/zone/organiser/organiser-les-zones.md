# Organiser les zones

Isocarto vous permet de structurer vos zones dans la barre latérale en créant des dossiers personnalisés, facilitant l’organisation et la navigation.

<VimeoVideo videoId="1165767479" />

## Créer des dossiers

Pour organiser vos zones, allez dans l’onglet "Zones dans la carte", puis cliquez sur l’icône de paramètres (roue dentée) appelée "Organiser par dossiers".

![Organiser par dossiers](/img/organiser-1.jpg)

Vous pouvez aussi créer et gérer les dossiers directement depuis le menu contextuel de chaque zone (les 3 points verticaux).

![Organiser par dossiers menu](/img/organiser-5.jpg)

Une fenêtre modale s’ouvrira.

Dans cette modale, vous pourrez créer des dossiers en les nommant, puis glisser vos zones existantes dans ces dossiers.

## Réorganiser les zones

Une fois les dossiers créés (cliquez sur le bouton "Nouveau dossier"), vous pouvez réorganiser les zones en les déplaçant dans ces dossiers ou entre eux. Survolez le nom d’un dossier pour modifier son titre avec l’icône de crayon.

![Organiser par dossiers](/img/organiser-2.jpg)

Glissez et déposez simplement les zones dans les dossiers désirés:

![Organiser par dossiers](/img/organiser-3.jpg)

Chaque dossier affiche un nombre total de zones qu’il contient.

:::tip[Info]

Le dossier intitulé "Sans dossier" est le dossier par défaut il ne peut pas être supprimé, mais s'il est vide il ne sera pas affiché!

:::

## Sauvegarder les modifications

Après avoir disposé les zones, cliquez sur "Enregistrer les modifications" pour appliquer vos changements. Vous serez ensuite ramené à la carte, où vous verrez la nouvelle organisation dans la barre latérale.

Vous pouvez réduire ou agrandir chaque dossier pour une meilleure lisibilité. Cela permet de simplifier l’interface et de mieux gérer vos espaces de travail.

![Organiser par dossiers](/img/organiser-4.jpg)
