# Création d'une zone manuelle

La zone manuelle permet de définir une aire géographique en dessinant directement sur la carte, sans contrainte de temps ou de distance.

## Étapes de création

<VimeoVideo videoId="1143873944" />

1. **Ouvrir le panneau "Ajouter nouvelle zone"**

2. **Sélectionner le type de zone** : Choisissez l’icône **"Manuelle"**.

3. **Activer le mode dessin** : Par défaut, le mode dessin est désactivé (bouton rouge). Cliquez sur le bouton pour l’activer (bouton vert).

4. **Dessiner la zone** : Cliquez sur la carte pour placer le premier point (un point jaune apparaîtra). Continuez à cliquer pour ajouter d’autres points et former un polygone.

5. **Valider la zone** : Une fois le polygone complété, vous pouvez :

   - Cliquer sur le dernier point jaune pour valider,
   - Appuyer sur la touche **Entrée**,
   - Faire un double-clic pour finaliser le polygone.

6. **Éditer la zone** : Si la forme ne vous convient pas, vous pouvez cliquer sur les points jaunes ou noirs pour ajuster le polygone.

7. **Finaliser la zone** : Une fois satisfait, cliquez sur **"Créer la zone"**. Si nécessaire, vous pouvez recommencer en utilisant le bouton **"Recommencer"**.

## Exemples d'utilisation

Les zones manuelles sont idéales pour :

- Définir des zones précises et personnalisées (ex : zones de protection, zones de restriction).
- Adapter la forme de la zone aux besoins spécifiques d’un projet.
