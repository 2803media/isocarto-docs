# Découpage administratif

Cette couche permet de visualiser les différents niveaux de découpage administratif utilisés en France.

<VimeoVideo videoId="1146634609" />

## Niveaux de découpage

Les différents niveaux administratifs sont disponibles, allant du plus petit au plus grand :

- **IRIS** : les quartiers
- **Communes**
- **EPCI** : Établissements Publics de Coopération Intercommunale
- **Départements**
- **Régions**

![visualisation](/img/decoupage-administratif.jpg)

## Utilisation de la couche

Pour afficher un découpage administratif :

1. **Sélectionner le niveau** : Choisissez le découpage souhaité (IRIS, commune, etc.).
2. **Activer les options d'affichage** : Vous pouvez afficher ou masquer les limites, les remplissages et les noms des zones administratives.

## Optimisation de la carte

Cette couche permet d’adapter la carte selon vos besoins, en affichant uniquement les niveaux de découpage pertinents pour votre analyse.
