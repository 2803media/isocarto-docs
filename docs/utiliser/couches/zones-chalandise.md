# Zones de chalandise

L’onglet **Zones de chalandise** permet de visualiser, modifier et de gérer les différentes zones sur la carte.

<VimeoVideo videoId="1146616412" />

## Gestion des zones de chalandise

- **Affichage global** : Utilisez le toggle principal pour afficher ou masquer toutes les zones de chalandise sur la carte.
- **Limites des zones** : Vous pouvez masquer ou afficher les limites des zones pour une meilleure clarté.
- **Remplissage des zones** : Ajustez la visibilité du remplissage des zones pour distinguer les différentes aires.
- **Marqueurs** : Les marqueurs peuvent être masqués ou affichés pour focaliser l’attention sur les zones.
- **Zone active** : Il est possible de masquer les limites de la zone active afin de mieux visualiser les zones environnantes.

Ce système de gestion permet une personnalisation fine des données affichées, facilitant ainsi l’analyse des zones commerciales et leur influence sur le territoire.
