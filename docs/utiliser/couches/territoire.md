# Territoire, environnement et risques

Dans l’onglet **Couches de données** de l’application Isocarto, la section **Territoire, environnement et risques** permet d’analyser les composantes physiques, fonctionnelles et les contraintes d’un territoire.

Ces couches sont essentielles pour contextualiser une zone d’étude avant un projet d’aménagement, d’implantation ou d’analyse stratégique.

<VimeoVideo videoId="1147027334" />

## Couches disponibles

Les couches suivantes peuvent être affichées sur la carte :

- **Inventaire biophysique**
- **Registre parcellaire graphique**
- **Réseaux de transport**
- **Taux de cambriolage**
- **Risques d’inondation**

Chaque couche peut être activée ou désactivée indépendamment.

## Inventaire biophysique

L’**inventaire biophysique** permet de visualiser les composantes naturelles du territoire.

![inventaire-biophysique](/img/inventaire-biophysique.jpg)

### Données affichées

Cette couche met en évidence :

- les **espaces naturels et agricoles**,
- les **zones boisées**,
- les **milieux humides**,
- les **éléments paysagers structurants**.

### Utilisation

Elle permet notamment de :

- comprendre la structure environnementale d’un territoire,
- identifier les zones sensibles ou à préserver,
- intégrer les contraintes écologiques dans un projet d’aménagement.

## Registre parcellaire graphique (RPG)

Le **registre parcellaire graphique** permet d’afficher les parcelles agricoles déclarées.

![registre-parcellaire](/img/registre-parcellaire.jpg)

### Affichage sur la carte

Les parcelles agricoles sont représentées avec :

- leurs **limites**,
- leur **occupation déclarée**,
- leur **usage principal**.

### Cas d’usage

Cette couche est utile pour :

- identifier les zones agricoles actives,
- analyser la pression foncière,
- étudier la compatibilité d’un projet avec les usages existants.

## Réseaux de transport

La couche **Réseaux de transport** permet d’analyser l’accessibilité et la mobilité.

![transport](/img/transport.jpg)

### Types de réseaux affichés

Selon le niveau de zoom, la carte peut afficher :

- les **réseaux de bus**,
- les **réseaux de métro**

### Intérêt pour l’analyse territoriale

Cette couche permet de :

- évaluer l’accessibilité d’un site,
- analyser les flux de déplacement,
- croiser l’offre de transport avec les zones d’habitat ou d’activité.

## Taux de cambriolage

La couche **Taux de cambriolage** permet de visualiser les indicateurs de sécurité sur le territoire.

![cambriolage](/img/cambriolage.jpg)

:::warning[Attention]

Les données ne sont disponibles que pour les grosses agglomérations.
:::

### Mode de représentation

Les données sont généralement représentées :

- par **zones statistiques**,
- avec une **graduation de couleurs** selon le niveau de taux.

### Utilisation

Cette couche est particulièrement utile pour :

- l’analyse résidentielle,
- l’évaluation d’un secteur avant un projet immobilier,
- la compréhension des dynamiques locales de sécurité.

## Risques d’inondation

La couche **Risques d’inondation** issue des [TRI (Territoires à Risque Important d'inondation)](/comprendre/glossaire#tri-territoire-à-risque-important-dinondation) permet d’identifier les zones exposées aux aléas hydrauliques (inondation et submersion marine).

![inondation](/img/inondation.jpg)

### Zones de risques

Les zones sont classées selon leur **niveau d’exposition**, par exemple :

- zones à **aléa faible**,
- zones à **aléa moyen**,
- zones à **aléa fort**.

### Cas d’usage

Cette couche permet de :

- anticiper les contraintes réglementaires,
- sécuriser les projets de construction ou d’aménagement,
- intégrer les risques naturels dans l’analyse territoriale.

## Cas d’usage

Les couches **Territoire, environnement et risques** sont particulièrement adaptées pour :

- analyser les contraintes naturelles et environnementales,
- anticiper les risques avant un projet immobilier ou urbain,
- croiser accessibilité, usages et risques,
- appuyer les décisions d’implantation ou de planification,
- enrichir les études territoriales et environnementales.

Elles offrent une **lecture globale et stratégique** du territoire, complémentaire aux données foncières, réglementaires et socio-économiques disponibles dans Isocarto.
